Live Online Class

Start or join a live class directly from your website.

This page uses embedded Jitsi Meet so teachers and students can launch a live class in-browser without a backend, database, React, or Node setup.

  • Teachers can generate a room code automatically
  • Students can join using a shared room code and password
  • Everything works inside one clean mobile-friendly page

Teacher should join first and share room code/password manually.

How It Works

Simple no-backend live class starter

Start Class Create a room from teacher details and launch the embedded meeting.
Join Class Students enter a room code and optional password to join the live session.
Reusable JavaScript The page uses small plain JavaScript functions for room management and meeting control.

Current Limitations

  • This is a no-backend starter setup.
  • Room approval is manual.
  • Room code and password are controlled and shared manually by you.
  • No secure login system is included yet.
  • A backend can be added later when leads grow.
